El box model es el algoritmo a través del cual el navegador dibuja las cajas en la pantalla. Solo se aplica a los elementos de bloque y cuenta con varios componentes:

  • Contenido → content-box
  • Separación entre el contenido y el borde → padding-box
  • Borde → border-box

También es necesario mencionar el margen, que es una propiedad externa e indica la separación entre cajas. La propiedad es margin. Ejemplo:
