Selectores
Selectores sencillos

1. Selector universal

Selecciona todos los elementos del documento web, se representa por:

*

2. Selectores de tipo o elemento

Es el selector más sencillo, ya que simplemente es una etiqueta de html.

Selector de clase

Una clase es un atributo de un elemento o etiqueta de html o xml.
Este es el más recomendable para dar estilo ya que puede ser reutilizado.

Selector por ID

Un ID es un atributo único que se asigna a un solo elemento.
Se usa principalmente en JavaScript y no debe repetirse en el documento.
Se representa con el símbolo #.
Selectores compuestos

Selector compuesto

Este tipo de selector combina varios selectores para aplicar estilos más específicos.
Ejemplo: div.clase selecciona todos los <div> que tengan esa clase.

Selector agrupado

Todos los selectores estan agruados por elemento caracter.

Selectores descendientes

indican cuando un elemento es hijo, nieto,etc.se agrupa con un espacio ejemplo.
